### В этом проекте мы:
- Создадим проект по обмену публичыми и приватными git-ами (кусками кода) на Rails
- Используем bootstrap, simple_form, slim, coderay
- Сделаем авторизацию пользователя руками через has_secure_password
- Выложим для просмотра на хостинг heroku https://gist-app-demo.herokuapp.com и репозиторий на github https://github.com/apavlyut/gist-app-demo
